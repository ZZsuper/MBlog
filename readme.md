# MBlog
***
这是个自己写着玩的Blog,刚接触前端没多久,练手之作
## 使用的东西
* Node.Js with Express
* AngularJs
* AngularMaterial
* Redis
* MongoDB
* Upyun

## 目前实现了的功能

### 面向外部
1.文章查看,评论 2.分享(正在完善自动抓取)

### 面向管理
1.文章管理 2.评论管理 3.分类管理
